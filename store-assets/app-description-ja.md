# 時差時計 - アプリストア説明文（日本語）

## アプリ名
時差時計

## サブタイトル（30文字以内）
世界の時刻を一目で確認

## 説明文

### 短い説明（80文字以内）
世界中の都市の現在時刻と時差を簡単に確認。海外の友人や取引先との連絡に最適なアプリ。

### 詳細説明

世界463都市の現在時刻をリアルタイムで確認できる時差時計アプリです。

【主な機能】
• 世界時計：お気に入りの都市を登録して、現地時刻を一覧表示
• 時差計算：2都市間の時差を瞬時に計算
• 直感的なUI：ダークテーマで目に優しいデザイン

【こんな方におすすめ】
• 海外の友人や家族と連絡を取る方
• 海外出張や旅行が多い方
• 国際的なビジネスをされている方
• 海外のイベントやスポーツ中継の時間を確認したい方

【対応地域】
アジア、ヨーロッパ、北米、南米、アフリカ、オセアニアの主要463都市に対応。

【特徴】
• 完全無料・広告なし
• オフライン対応
• プライバシー重視（データ収集なし）
