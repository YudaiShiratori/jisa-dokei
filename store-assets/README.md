# アプリストア素材 / Store Assets

このディレクトリには、App Store / Google Play 申請用の素材が含まれています。

## ファイル一覧

| ファイル | 説明 |
|---------|------|
| `app-description-ja.md` | 日本語アプリ説明文 |
| `app-description-en.md` | 英語アプリ説明文 |
| `privacy-policy.md` | プライバシーポリシー（日英） |

## アイコン・画像

アプリアイコンとスプラッシュスクリーンは `assets/images/` に配置されています：

| ファイル | サイズ | 用途 |
|---------|-------|------|
| `icon.png` | 1024x1024 | App Store / Google Play アイコン |
| `adaptive-icon.png` | 1024x1024 | Android Adaptive Icon |
| `splash-icon.png` | 1024x1024 | スプラッシュスクリーン |
| `favicon.png` | 48x48 | Web用ファビコン |

## スクリーンショット

スクリーンショットは以下のサイズで用意してください：

### iOS
- iPhone 6.7" (1290 x 2796)
- iPhone 6.5" (1284 x 2778)
- iPhone 5.5" (1242 x 2208)
- iPad 12.9" (2048 x 2732)

### Android
- Phone (1080 x 1920 以上)
- 7" Tablet (1200 x 1920)
- 10" Tablet (1800 x 2560)

## Feature Graphic (Android)

Google Play用のFeature Graphicは 1024 x 500 ピクセルで作成してください。
